### `addMarkup`

Add one or more markup entities using XML encoded text

**Parameters**

-   `makupxml`: String in xml format that holds tags in accordance with RxView360 xml based markup format

Returns NA

**Syntax**

```javascript
RxCore.addMarkup(markupxml);
```
