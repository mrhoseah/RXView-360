### `blockhoverevent`

Use to activate or turn off GUI_2DBlockInfoPos callback when moving mouse over 2D vector polygons.

Returns NA

**Syntax**

```javascript
RxCore.blockhoverevent(onoff);
```

**Parameters**

-   `onoff`: Boolean - true = on, false = off.
